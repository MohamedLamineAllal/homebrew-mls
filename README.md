# Homebrew Tap for MacosLeanStorage (mls)

This is the Homebrew tap for [MacosLeanStorage](https://github.com/MohamedLamineAllal/MacOSLeanStorage).

## Installation

You can install `mls` by tapping this repository:

```bash
brew tap MohamedLamineAllal/mls
brew install mls
```

## Usage

After installation, you can run `mls` directly from your terminal:

```bash
mls --help
```
